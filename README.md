# 数据导入验证工具 - Data Import Validator

## 项目概述

数据导入验证工具是一个综合性解决方案，旨在解决客户在数据录入/导入过程中遇到的格式规范问题，减少对人工支持的依赖。该工具提供实时格式验证、自动修复、规范文档、问题排查指南和视频教程等功能。

## 核心功能

### 1. 数据导入验证
- **多格式支持**：Excel (.xlsx, .xls) 和 CSV 文件
- **智能识别**：自动检测字段类型（日期、数字、编码等）
- **实时验证**：上传后立即显示验证结果
- **自动修复**：自动转换日期、数字等常见格式问题
- **详细报告**：生成包含错误位置、建议和修复方案的详细报告

### 2. 模板校验
- **多种模板**：员工信息、产品信息、订单数据、库存数据
- **字段说明**：每个字段的类型、必填性、示例值
- **模板下载**：下载空白模板和示例数据
- **字段验证规则**：实时展示字段规范要求

### 3. 规范文档
- **日期格式规范**：多种日期格式的正确与错误示例
- **数字格式规范**：整数、小数、金额的格式要求
- **文本格式规范**：特殊字符处理和编码问题
- **编码格式规范**：各类编码的命名规则
- **导入指南**：文件要求、编码指南、批量导入流程

### 4. 问题排查指南
- **日期格式问题**：5步排查流程
- **数字格式问题**：常见格式错误和解决方案
- **编码问题**：乱码处理和编码转换
- **必填字段缺失**：空值检查和数据完整性验证
- **重复数据**：主键冲突检测
- **关联数据错误**：外键约束验证

### 5. 视频教程
- **快速入门**：数据导入基础（5:30）
- **日期格式规范详解**：日期录入方法（8:45）
- **常见问题排查实战**：实际案例演示（6:20）
- **模板使用指南**：正确填写模板（4:15）

## 支持的验证类型

| 类型 | 说明 | 示例 | 自动修复 |
|------|------|------|----------|
| date | 日期 | 2024-01-15 | ✅ |
| datetime | 日期时间 | 2024-01-15 14:30:00 | ✅ |
| integer | 整数 | 123 | ✅ |
| decimal | 小数 | 123.45 | ✅ |
| currency | 金额 | 1234.56 | ✅ |
| phone | 电话 | 13800138000 | ❌ |
| email | 邮箱 | example@domain.com | ❌ |
| idcard | 身份证 | 110101199001011234 | ❌ |
| code | 编码 | PROD_2024_001 | ❌ |

## 自动修复能力

### 日期格式修复
- `2024/01/15` → `2024-01-15`
- `2024年01月15日` → `2024-01-15`
- `20240115` → `2024-01-15`
- `01/15/2024` → `2024-01-15`
- Excel序列号日期（如45242）→ `2023-10-29`

### 数字格式修复
- 移除千分位分隔符：`1,234,567` → `1234567`
- 移除货币符号：`¥1000` → `1000`
- 修复小数位数：`123.456` → `123.46`
- 移除文本内容：`约100` → `100`

## 快速开始

### 1. 本地运行
```bash
# 进入项目目录
cd data-import-validator

# 启动本地服务器（使用Python）
python -m http.server 8000

# 或使用Node.js
npx serve .

# 打开浏览器访问
http://localhost:8000
```

### 2. 使用验证工具
1. 打开网页，进入"数据验证"标签
2. 拖拽或点击上传Excel/CSV文件
3. 系统自动检测字段类型并验证
4. 查看验证结果，下载修复后的文件

### 3. 下载模板
1. 进入"模板校验"标签
2. 选择需要的模板类型（员工、产品、订单、库存）
3. 查看字段说明和示例
4. 点击"下载空白模板"

### 4. 查阅规范
1. 进入"规范文档"标签
2. 使用左侧导航或搜索功能
3. 查看详细的格式规范和示例

### 5. 问题排查
1. 进入"问题排查"标签
2. 选择遇到的问题类型
3. 按照步骤逐一排查
4. 使用验证工具确认修复
